How to Run a Download Speed Test

1. Pause Other Traffic

Stop downloads, streaming, and backups so your download speed test is accurate.

2. Use a Nearby Server

Choose the closest test server for a better measure of real download speed.

3. Test on Ethernet

4. Run Multiple Tests
